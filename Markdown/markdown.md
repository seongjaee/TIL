## 내용

이것은 내용이다.

**파이썬 리스트**는 `[]`로 *만듭니다*.

```python
a = [1, 'b', '사과']
a.sort()
```



내용을 꾸미는 방법

볼드체 : * 두개

이탤릭체 : * 한개

인라인 코드 : ` 한개

코드 블럭 : ` 세개, 언어를 옆에 붙이면 더 예쁨
