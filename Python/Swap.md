# [교체 연산](https://ko.wikipedia.org/wiki/%EA%B5%90%EC%B2%B4_%EC%97%B0%EC%82%B0)

- 두 변수에 들어있는 값을 서로 맞바꾸는 연산.



## 방법 1 - 임시 변수 활용

```
temp = a
a = b
b = temp
```

- 가장 쉬움.
- 대부분 가장 효율적
- 변수의 메모리가 크면 안 좋을 수도.

## 방법 2 - XOR 이용

```
a = a ^ b
b = a ^ b
a = a ^ b
```

- 두 변수가 같은 메모리 값을 가지면 작동하지 않음



## 방법 3 - 덧셈 뺄셈 이용

```
a = a + b
b = a - b
a = a - b
```

- XOR보다 연산속도가 느림.
- 오버플로우 발생 가능성



## 방법 4

- 파이썬의 경우, 튜플을 이용함. :+1:

```python
a, b = b, a
```
